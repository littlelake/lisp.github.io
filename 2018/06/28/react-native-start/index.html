简单运行一个react-native项目

发表于 2018-06-28 | 分类于 javascript

不按照react-native中文网上的来的话，只需要五步就可以完成react-native的demo运行。

  1. 下载并安装jdk，以安装jdk8为例
  2. 配置环境变量
    （1）配置android的环境变量
    ANDROID_HOME — C:\Users\cyy\AppData\Local\Android\Sdk
    （2）配置java的环境变量
    JAVA_HOME — C:\Program Files\Java\jdk1.8.0_171
    （3）在path中配置platform-tools的环境变量
    PATH — C:\Users\cyy\AppData\Local\Android\Sdk\platform-tools
  3. 下载android-studio，解压后点击android-studio\bin\studio.exe，然后在android-studio中配置jdk可以根据中文网上的说明进行配置
  4. 运行device，点击android-studio右上角的AVD Manager，选择适当的手机模拟器设备，一般参见“Nexus 5X API 24” - “android 7.0”，然后点击运行就可以了。中文网上面用Genymotion 使用起来比较繁琐
  5. 初始化项目
    （1）react-native init android-init-name
    （2）cd android-init-name
    （3）react-native run-android

另外，如果一个项目在react-native run-android运行起来的时候报错，可以试着删除“android\build”和“android\app\build”后再运行。
